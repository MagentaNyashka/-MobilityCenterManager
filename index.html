Добро пожаловать в систему автоматизации Центра Мобилизации Метро!

Этот проект предназначен для автоматического распределения запросов людей с ограниченными возможностями между сотрудниками Центра Мобилизации.

Основные возможности:

  • Мониторинг сотрудников и заказов: Просмотр текущего статуса сотрудников и активных запросов.
  • Автоматическое распределение: Система автоматически назначает сотрудников на запросы, учитывая их местоположение и доступность.
  • Графическое представление данных: Удобный интерфейс для визуализации данных о сотрудниках, заказах и маршрутах.

Технологии:

  • Frontend: HTML, CSS (Bootstrap) для создания удобного интерфейса.
  • Backend: PHP для работы с базой данных MySQL.
  • Серверная логика: Node.js для выполнения алгоритмов распределения и обновления данных.

Как это работает:

  1. Система собирает данные о сотрудниках, заказах и станциях метро из базы данных.
  2. Алгоритм распределения назначает сотрудников на запросы, минимизируя время ожидания.
  3. Администратор может отслеживать статус выполнения запросов через веб-интерфейс.

Для кого это:

Для сотрудников Центра Мобилизации Метро, чтобы упростить управление запросами, и для администраторов, чтобы контролировать эффективность работы системы.
